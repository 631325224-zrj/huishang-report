# 2026-04-08 工作日志

## 问题排查：API数据错误（只返回每月1日）

### 问题现象
用户反馈报表数据与 Excel 标准数据不符：
- 报表 2月：709,877（只有1天的数据）
- 标准 2月：23,909,483（整月）

### 根本原因
`getBusinessSituation` API 的 `dateType='month'` 参数：
- ❌ 实际含义：查询该日期的业务数据（只返回当天）
- ✅ 正确做法：使用 `getserialdata` 账单明细 API

### 解决方案
添加 `queryMonthBillList()` 函数：
1. 调用 `GET /datatransfer/getserialdata` 
2. 按 `settle_biz_date` 过滤当月数据
3. 按 `income_total` 聚合纯收金额
4. 支持分页获取全部账单

### API 对比
| API | 用途 | dateType='month' 行为 |
|-----|------|----------------------|
| getBusinessSituation | 月度汇总 | ❌ 只返回每月1日 |
| getserialdata | 账单明细 | ✅ 可按业务日期聚合整月 |
