### 出租率计算完整规则（最终正确版）

```javascript
// 1. 预订数（getBookOrderDetailData）
list.forEach(b => {
  if (b.bookState !== '已执行') return;   // 白名单：只算已到店的
  const shift = b.shift || b.shift_name || '';
  if (shift.includes('午')) noon++;
  else if (shift.includes('晚')) evening++;
});

// 2. Walkin 包间数（getserialdata）
list.forEach(b => {
  if (String(b.order_type) === '2') return;  // 跳过预订单
  const icb = b.is_continued_bill;
  // 显式假值 → 非续单；其余全算续单排除
  if (icb != null && icb !== false && icb !== '' && icb !== 'false' && icb !== 'F' && icb !== 0 &&
      String(icb).trim().toLowerCase() !== 'false' && String(icb) !== '0') return;
  if (!isPrivateRoom(b.point_name)) return;  // 只算包间
  const shift = getShiftByOpenTime(b.open_time);  // <16 午市，≥16 晚市
  if (shift === 'noon') noon++;
  else if (shift === 'evening') evening++;
});

// 3. 包间判断
function isPrivateRoom(name) {
  if (!name) return false;
  if (/[0-9a-zA-Z]/.test(name)) return false;  // 排除含数字/字母的
  if (name.includes('外') || name.includes('盒') || name.includes('宴')) return false;
  return true;
}

// 4. 出租率公式
出租率 = (预订数 + Walkin包间数) / 总包间数
```

### 关键教训
1. **预订过滤必须用白名单**（`=== '已执行'`），不能用黑名单排除"取消"。API 还有其他状态如"已过期""待执行"等。
2. **续单判断字段只能是 `is_continued_bill`**，不能用 `is_continued_bill_name`（对所有账单都有值）。
3. **诊断日志三板斧**：状态分布统计 → 字段格式分析 → 逐条明细输出，缺一不可。
