### 问题：月度汇总数据与文档始终有细微差异
- 2月差 80.00 元
- 3月差 251.50 元

**根本原因（最终确认）**：
- `queryMonthBillList` 之前用的是 `getserialdata` API + `last_total - not_income_money`
- 正确方案：**月度汇总也用 `getBusinessSituation` API**（与日数据相同）
- 纯收字段：`income.incomeTotal`（不是 `last_total - not_income_money`）

**文档验证数据**：
- 朝阳门店 2026-02: **1,690,526.55** ✅
- 朝阳门店 2026-03: **1,753,382.10** ✅
- 朝阳门店 2026-01: **1,978,630.54** ✅

**教训**：`getserialdata`（账单明细）和 `getBusinessSituation`（日数据汇总）口径不同，不能混用
