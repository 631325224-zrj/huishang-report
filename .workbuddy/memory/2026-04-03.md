### 🎯 重要里程碑：纯收金额字段修复

**问题**：报表一直显示营收数据，而非纯收金额

**解决过程**：
1. 添加调试面板，逐步排查 API 返回字段
2. 发现 `income`、`incomeTotal`、`notIncomeTotal` 等字段
3. 用户提供完整 API 返回示例
4. 确认正确字段路径：
   - 纯收金额：`data.List.income.incomeTotal`
   - 非收入金额：`data.List.notIncome.notIncomeTotal`
   - 实收：`data.List.businessData.last`
   - 应收：`data.List.businessData.orig`

**验证公式**：
- 应收 - 优惠 = 实收
- 实收 - 非收入 = 纯收
